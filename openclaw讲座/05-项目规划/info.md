# 讲座内容摘要与设计需求

## 讲座主题
AI智能体技术讲座：从OpenClaw到企业落地

## 目标受众
中国移动内部非技术人员（30-45分钟线上会议）

## 核心内容结构（7大章节）
1. **开场**：从ChatGPT到Agent的认知跃迁（Copilot→Vibe Coding→Agentic时代）
2. **核心概念**：ReAct范式、MCP协议、Skills系统、多Agent协作
3. **OpenClaw详解**：三层架构、四层记忆、Skills生态、成功原因
4. **三Agent对比**：OpenClaw（通用）vs Hermes（自进化）vs Claude Code（编程）
5. **能力全景**：擅长/不擅长、进阶技巧、完整场景穿透（10步流程）
6. **移动场景**：模型底座、Skills源、浏览器自动化、4个场景穿透
7. **总结展望**：三个关键结论、超级个体理念

## 关键数据点
- OpenClaw: 278,932 Stars (GitHub全球第一), 13,729 Skills, 20+平台
- Hermes Agent: GEPA自进化, 0 CVE, 113ms记忆检索
- Claude Code: SWE-bench 87.6%, SOC 2认证, $25亿ARR
- 中国移动: 九天大模型, 17000+数字员工, 年减负70万人天
- 场景效率提升: 邮件30min→3min, 经营报告2-3天→30min, 故障诊断2-4h→10min, 数据稽核5天→1h

## 设计要求
- **风格**：干净、清爽、科技风，不要太极客
- **配色**：白色/浅灰底色 + 中国移动蓝(#0066CC或#0099FF)为主色
- **辅助色**：科技青(#00B4D8)、活力橙(#FF6B35)、浅蓝(#E8F4FD)
- **不要**：大面积黑色背景、死板PPT风格、过度极客风
- **要**：可折叠展开细节、翻页导航、动画效果、主题可切换
- **每页独立**：每个幻灯片页面是独立HTML，可单独编辑
- **模块化**：样式和配色分离，可配置多套主题
- **页面数**：18页（封面+目录+7章内容页+补充细节页+总结）
